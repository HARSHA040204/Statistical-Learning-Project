• NeuroPred-FRL
    Predictor for neuropeptide prediction









Input System:


FASTA format is required for input the protein sequences in our server. Only the 20 conventional amino acid symbols are supported.

Output System:


The prediction output consists of four items:
◦Peptide name name
Predicted score
◦Justification of annotation measures

Browser Tool:


The website has been tested with browsers of Internet Explorer 7 or later.
